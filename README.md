### 🧑‍💻 Who I Am

안정적인 **클라우드(AWS) 백엔드 서비스**부터 **데스크톱 클라이언트 애플리케이션**까지, 다양한 환경의 개발을 아우르는 백엔드 개발자 이용찬입니다. 
단순히 임무를 완수하는 것을 넘어, **새로운 지식 습득을 즐기고 이를 성장의 발판으로 삼아** 끊임없는 발전을 도모합니다.

`Nebula` 프로젝트에서는 **Spring Boot + AWS EC2**로 백엔드를 배포해 실제 클라우드 환경을 경험했고,  
`Nebula-Client`에서는 **FastAPI**로 로컬 파일 시스템을 다루는 데스크톱 클라이언트를 구현하며 목적에 맞는 기술 선택과 응용력을 익혔습니다.

또한 zeroton 대회 `Rentree` 프로젝트를 통해 **코드 리뷰와 의견 조율**의 가치를 체감하며,  
혼자가 아닌 팀으로 함께 문제를 해결하는 개발자로 성장하고 있습니다.
