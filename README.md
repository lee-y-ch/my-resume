## 💡 핵심 역량 (Core Competencies)

* **백엔드 시스템 설계:** **Spring Boot (Java)**를 활용한 비즈니스 로직 구현 및 `[JPA/MyBatis]` 기반 데이터베이스 연동 경험(`Tabling`, `Nebula-Backend`)과 **FastAPI (Python)**를 이용한 고성능 비동기 RESTful API 서버 설계 경험(`Nebula-Client`)을 보유하고 있습니다.
* **프론트엔드 개발 (New):** **React** 및 **TypeScript**를 활용하여 `[컴포넌트 기반 UI/UX 설계]` 및 `[API 연동]`을 수행한 경험 (`Rentree` 프로젝트)
* **클라우드 인프라 (DevOps):** **AWS**의 `[EC2, S3 등]`를 활용하여 `Nebula-Backend` 애플리케이션을 배포 및 운영한 경험이 있습니다. 또한 **Docker**를 이용한 컨테이너화, **Jenkins**를 활용한 `CI/CD` 파이프라인 구축 경험이 있습니다.
* **IoT 시스템 연동:** `[Raspberry Pi, Python]` 등을 활용해 `[센서 데이터 수집]` 및 제어 시스템을 구축(`EmbeddedIoT`)하고, 백엔드 서버와 데이터를 연동한 경험이 있습니다.
* **협업 및 버전 관리:** **Git & GitHub**를 기반으로 `Rentree` 등 여러 프로젝트에서 **공동 작업(Collaborator)**을 수행하며 코드 리뷰, 브랜치 관리, 이슈 트래킹을 경험했습니다.
